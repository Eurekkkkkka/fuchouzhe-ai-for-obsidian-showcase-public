# 项目提交说明：富酬者 AI for Obsidian Showcase

## 项目名称

富酬者 AI for Obsidian Showcase

## 项目链接

https://github.com/Eurekkkkkka/fuchouzhe-ai-for-obsidian-showcase-public

## 一句话介绍

一个可安装体验的 Obsidian 本地 AI 工作流插件展示版，让 AI 能读取笔记、理解文档、调用 Vault 工具，并把结果写回知识库。

## 项目解决的问题

普通 AI 聊天工具很难持续理解用户本地知识库，用户需要频繁复制粘贴笔记、文档和资料。富酬者 AI 尝试把大模型能力放进 Obsidian，让 AI 直接进入个人知识库和日常工作流，帮助用户整理资料、解析文档、生成脚本、规划任务和沉淀笔记。

## 我的角色

我负责项目定位、功能规划、插件原型推进、AI 协作开发、落地页、购买页、API 配置教程和交付材料整理。

## 核心能力

- Obsidian 内 AI 对话。
- 当前笔记和选中文本上下文读取。
- Markdown / Word / Excel 文件解析。
- Vault 文件读取、搜索、写入和整理。
- 场景动作面板：开始一天、AI 资讯、今日财经、归档整理、爆款文案、改善开头、去 AI 味、对标探索、卡片制作、写作引擎。
- Skills 工作流加载。
- MiniMax MCP 联网搜索和图片理解。
- 公开展示版 Demo Mode：移除商业手机号登录，安装后可直接进入插件体验。

## 使用的 AI 工具

- ChatGPT / Claude：需求拆解、产品表达、文档整理。
- Claude Code / Codex：代码理解、功能调整、README 与交付材料整理。
- Agent Skills：把重复工作流拆成可复用的任务模板。
- MiniMax API：插件内 AI 对话和 MCP 能力。

## 技术栈

- Obsidian Plugin API
- TypeScript
- esbuild
- MiniMax API
- MiniMax MCP
- Markdown / Word / Excel parsing
- Local file workflow

## 公开版说明

这个仓库是可公开、可安装的 Showcase 版本。它保留插件主体体验，但不包含商业版中的手机号登录、付费授权、服务端白名单、私有 API 地址和用户交付配置。

这样既方便 HR 和面试官实际安装体验，也不会影响未来把完整商业版作为付费产品继续变现。
